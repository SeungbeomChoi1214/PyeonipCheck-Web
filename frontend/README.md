# 편입 합격 예측 대시보드

편입 준비생들을 위한 데이터 기반 합격 가능성 진단 시스템입니다.

## 🚀 실행 방법

### 1. 백엔드 서버 실행
```bash
# 프로젝트 루트에서
python main.py
```

### 2. 프론트엔드 실행
```bash
# frontend 폴더에서
cd frontend
npm start
```

## 📊 주요 기능

### 1단계: 대학/학과 선택
- 전국 대학 목록에서 선택
- 선택한 대학의 학과 목록 표시
- 인문계/자연계 구분 표시

### 2단계: 점수 입력
- 영어 점수 입력 (필수)
- 수학 점수 입력 (자연계만)
- 실시간 유효성 검사

### 3단계: 결과 대시보드
- **종합 판정**: 안정/적정/소신/우주상향
- **위치 시각화**: 전체 경쟁자 중 나의 위치를 그래프로 표시
- **상세 통계**: 등수, 백분위, 점수 정보

## 🎯 UX 특징

- **편입생 심리 반영**: 단순 등수가 아닌 합격 가능성 중심의 정보 제공
- **시각적 대시보드**: 차트와 게이지를 활용한 직관적 정보 전달
- **반응형 디자인**: 모바일에서도 편리한 사용
- **애니메이션 효과**: 결과 표시 시 임팩트 있는 연출

## 🛠 기술 스택

- **Frontend**: React, Framer Motion, Recharts, Axios
- **Backend**: FastAPI, SQLAlchemy, SQLite
- **Design**: CSS3, Responsive Design

## 📱 화면 구성

1. **메인 헤더**: 시스템 제목과 설명
2. **단계 표시기**: 현재 진행 단계 시각화
3. **카드 기반 UI**: 각 단계별 깔끔한 카드 레이아웃
4. **결과 대시보드**: 
   - 종합 판정 카드 (상단)
   - 위치 시각화 카드 (중간)
   - 상세 데이터 카드 (하단)
