# 편입플래너 개발 명세서

## 1. 점수 산출 로직 (Backend)
- 파일 위치: backend/services.py
- 공식: (영어점수 * 0.5) + (수학점수 * 0.5) = 100점 만점
- 절대로 단순히 더해서 200점이 나오면 안 됨.

## 2. 합격 예측 메시지 기준
- Gap >= 0: 안정
- -10 <= Gap < 0: 적정
- Gap < -25: 위험

## 3. 유료화 정책 (Frontend)
- 파일 위치: frontend/src/components/PredictPage.js
- isPaidUser가 false이면 상세 결과 div에 'blur-sm' 클래스 적용

## 4. 점수 산출 공식
A. 점수 산출 공식 (Normalization)어떤 대학, 어떤 전형이든 결과값은 무조건 100점 만점으로 통일합니다. 

- 내 환산 점수 = (내 영어 점수 * 영어 반영비) + (내 수학 점수 * 수학 반영비) + (내 학점 환산 * 학점 반영비)

조건: 반영비의 합(영어+수학+학점)은 무조건 1.0 (100%)이어야 함.

예시 (성균관대 자연계 - 영어 50%, 수학 50% 가정):내 점수: 영어 70점, 수학 80점

계산: (70 * 0.5) + (80 * 0.5) = 35 + 40 = 75점

절대 합산(70 + 80 = 150) 하지 말 것.

B. 합격 예측 판정 로직 (The Verdict)사용자의 점수(My Score)와 목표 대학의 작년 합격 컷(Cutline)의 차이(Gap)에 따라, 기계적인 4단계 판정만 내립니다. 

AI에게 "희망적인 말 해줘"라고 시키지 마세요.Gap = 내 점수 - 합격 컷구간 (Gap)판정 (Status)사용자에게 보여줄 메시지 (Message)색상 (UI)+0.1점 이상안정 (Safe)"합격권입니다. 

Gap = 내 점수 - 합격 컷

구간 (Gap),판정 (Status),사용자에게 보여줄 메시지 (Message),색상 (UI)
+0.1점 이상,안정 (Safe),"""합격권입니다. 실전 감각 유지에 집중하세요.""",🟢 초록
-5점 ~ 0점,적정 (Proper),"""추가합격 사정권입니다. 한 문제 차이입니다.""",🟡 노랑
-15점 ~ -5.1점,소신 (Challenge),"""상향 지원입니다. 남은 기간 전략 과목에 올인하세요.""",🟠 주황
-15.1점 이하,위험 (Danger),"""현재로서는 합격 확률이 희박합니다. 눈높이 조정이 필요합니다.""",🔴 빨강
